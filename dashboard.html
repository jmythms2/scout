Major Assumptions

Assumptions are a necessary element of computational models. Assumptions make it possible to create a model that can be solved quickly and with limited computational resources. Because name models future resource demands from residential and commercial buildings, even a complicated and detailed model with as few assumptions as possible might not be an accurate reflection of the future. Selecting appropriate assumptions balances model accuracy with computational complexity and execution time.
